# Auto Chat Reply Bot

This is a **Python automation project** that reads chat messages from the screen, generates a reply using **Google Gemini AI**, and sends the reply automatically.

---

## What it does

* Reads chat text from the screen
* Checks who sent the last message to avoid replying to self
* Uses Gemini AI to generate a desired reply
* Pastes and sends the reply automatically

---

## Built With

* Python
* pyautogui
* pyperclip
* Google Gemini API

---

## Important Notes

* Screen coordinates are fixed (may need adjustment)
* Uses screen automation, not official chat APIs
* Gemini free tier has usage limits
* Project is for educational use only

---

## Models Used

* gemini-2.5-flash-lite
* gemini-2.5-flash-pro
